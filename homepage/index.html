The Path Expression Templates Library

PET is a library that implements generic regular path expressions. It provides a set of classes and overloaded operators that allow the programmer to define regular path expressions over user-defined data structures using solely the GCC compiler.

Documentation

The main features of the library are documented in the paper Compilation of Generic Regular Path Expressions Using C++ Class Templates which appears in the proceedings of the 14th Conference on Compiler Construction, LNCS 3443, pages 27-42, Springer, 2005. The library has a rather complex implementation but the end user should be only concerned with the high-level classes and operators. Check the source code, in particular the pet.hh file and the test files for usage examples.

Please contact the author for questions and information about the library.
